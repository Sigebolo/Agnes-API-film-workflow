# MFilm-CLI 长视频链式生成功能

## 原理

Agnes API 以参考图片为起始帧生成视频。利用这个特性：
1. 生成视频 N → 提取末帧截图 → 作为视频 N+1 的参考图 → 新提示词
2. 循环往复，生成多段连贯视频
3. 最后 ffmpeg concat 拼接成完整长视频

**关键约束：** 提示词必须与当前参考图的场景逻辑一致（图片在沙发看书 → 不能提示"洗澡"，会生成沙发上出水）

## 新增命令

### `mfilm chain` — 长视频链

```bash
# 方式1: JSON 文件（复杂剧情）
mfilm chain --scenes scenes.json

# 方式2: 命令行多 prompt（简单串联）
mfilm chain --prompt "场景1" --prompt "场景2" --prompt "场景3"

# 可选参数
--initial-image "https://..."    # 首段参考图（不指定则文生视频）
--dna "Maoning"                  # 使用 DNA 预设
--duration 15                    # 每段时长（默认15s）
--output-dir "./chain_output"    # 输出目录
--no-concat                      # 不自动拼接
--api-key "sk-xxx"
```

### JSON 格式

```json
{
  "initial_image": "https://...",
  "dna": "Maoning_Standard",
  "output_dir": "./chain_output",
  "no_concat": false,
  "scenes": [
    {"prompt": "女孩在沙发上看书，微笑", "duration": 15, "label": "scene_01"},
    {"prompt": "女孩放下书，站起来伸懒腰", "duration": 10, "label": "scene_02"},
    {"prompt": "女孩走向窗边，阳光洒在脸上", "duration": 15, "label": "scene_03"}
  ]
}
```

## 核心流程

```
for each scene:
  1. load reference_image (第一段用 initial_image，后续用上一段末帧)
  2. enhance_prompt(scene.prompt + DNA traits)
  3. submit_task(prompt, reference_image) → video_id
  4. poll_task(video_id) → video_url
  5. download_video(video_url) → scene_N.mp4
  6. extract_last_frame(scene_N.mp4) → scene_N_lastframe.jpg
  7. reference_image = base64_data_url(scene_N_lastframe.jpg)

after all scenes:
  if not --no-concat:
    ffmpeg concat all scene_*.mp4 → final_long_video.mp4
```

## 新增函数

### `extract_last_frame(video_path) -> str`
```python
def extract_last_frame(video_path: str) -> str:
    """Extract last frame from video, return base64 data URL for API reuse."""
    import subprocess
    frame_path = video_path.rsplit(".", 1)[0] + "_lastframe.jpg"
    subprocess.run([
        "ffmpeg", "-sseof", "-1", "-i", video_path,
        "-frames:v", "1", "-q:v", "2", frame_path, "-y"
    ], capture_output=True, check=True)
    
    with open(frame_path, "rb") as f:
        b64 = base64.b64encode(f.read()).decode()
    return f"data:image/jpeg;base64,{b64}"
```

### `concat_videos(video_paths, output_path) -> bool`
```python
def concat_videos(video_paths: list, output_path: str) -> bool:
    """Concatenate multiple videos using ffmpeg."""
    import subprocess
    list_file = output_path + ".txt"
    with open(list_file, "w") as f:
        for p in video_paths:
            f.write(f"file '{os.path.abspath(p)}'\n")
    result = subprocess.run([
        "ffmpeg", "-f", "concat", "-safe", "0",
        "-i", list_file, "-c", "copy", output_path, "-y"
    ], capture_output=True)
    os.remove(list_file)
    return result.returncode == 0
```
