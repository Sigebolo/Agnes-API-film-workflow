# 广告视频生成工作流

## 核心原则

**视频画面生成和TTS配音生成是完全独立的两条线**

```
视频画面生成线：
商品处理 → 脚本创作 → 分镜设计 → 视频Prompt → 生成视频片段 → 拼接视频

TTS配音生成线（独立进行）：
脚本创作 → TTS配音生成 → 添加TTS到视频

两线汇合：
视频拼接与音频合成 = 拼接好的视频 + 整段TTS音频 + BGM → 最终广告视频
```

## 操作步骤

### 第一阶段：商品处理与信息识别

1. **商品主体扣图**
   - 使用 rembg 自动去除背景，提取透明PNG商品图

2. **商品信息识别**
   - 商品类型（服装、数码、家居、食品等）
   - 主要特征（颜色、材质、设计风格）
   - 卖点信息（功能、优势、适用场景）
   - 品牌调性（高端、亲民、潮流、专业等）

3. **信息补充询问**
   - 目标受众（年龄、性别、消费习惯）
   - 广告时长（15s、30s、60s）
   - 视频风格（简约、时尚、科技感、温馨等）
   - 核心卖点（需重点突出的1-3个特点）

### 第二阶段：创意设计与脚本生成

4. **广告脚本创作**
   - 开场（3-5秒）：吸引注意力，展示品牌/产品概念
   - 展示（5-15秒）：多角度展示商品，突出核心卖点
   - 卖点（5-10秒）：深入讲解优势和使用场景
   - 结尾（3-5秒）：品牌信息、购买引导

5. **分镜设计**
   - 为脚本每个段落设计详细分镜
   - 每个分镜包含：时长、画面描述、数字人位置、商品展示方式

6. **生成视频Prompt**
   - ⚠️ Prompt只包含视觉元素，不包含台词
   - ⚠️ 商品主体必须100%保持原貌
   - 参考 video-prompt-guide.md

### 第三阶段：视频生成

7. **并行生成视频片段**
   - 为每个分镜调用视频生成API
   - Prompt只包含视觉元素（画面、运镜、打光、商品、场景）

8. **视频片段收集**
   - 按分镜顺序整理为列表
   - 检查视频质量

### 第四阶段：TTS配音生成（必须在视频生成完成后）

9. **TTS配音生成**
   - ⚠️ 必须在视频片段生成完成后进行
   - 基于完整脚本的整段台词一次性生成单个TTS音频文件
   - 语速：280-320字/分钟
   - 整段TTS保持均匀一致的语速

### 第五阶段：后期合成

10. **BGM选择**
    - 根据视频风格推荐BGM类型

11. **视频拼接与音频合成**
    - 按顺序拼接所有视频片段
    - 将整段TTS配音添加到拼接后的视频
    - 叠加BGM背景音乐
    - 调整音频混音平衡

12. **质量检查与交付**
    - 视频流畅度（转场是否自然）
    - 音画同步（TTS与画面匹配）
    - 音频清晰度（人声与BGM平衡）
    - 整体时长是否符合要求

## 关键约束

### 视频Prompt约束
- Prompt只包含视觉元素，绝对不包含台词
- 商品主体100%保持原貌
- 通过运镜、打光、场景增强展示，而非改变商品本身
- 推荐运镜：往前推、旋转、平移、环绕、静止

### TTS约束
- 基于完整脚本的整段台词一次性生成
- 禁止按分镜分段生成再拼接
- 语速均匀一致：280-320字/分钟
- 音频时长与视频总时长匹配（±1秒）
